# [[XYPosition]]

## 개요
XYPosition Object는 2D POSITION의 X 및 Y 구성요소를 나타내는 객체이다. 주어진 좌표에서 XYPOSITION을 생성한다.

## 멤버 (Members)

| Name | Type | Purpose |
| :--- | :--- | :--- |
| **X** | [[REAL\|REAL Object]] | 2D POSITION의 X 구성요소. |
| **Y** | [[REAL\|REAL Object]] | 2D POSITION의 Y 구성요소. |

## 메소드 (Methods)

| Name | Result | Purpose |
| :--- | :--- | :--- |
| **XYposition** () | XYPOSITION | 주어진 좌표에 XYPOSITION을 생성한다. |
| **String** () | [[STRING\|STRING Object]] | XYPOSITION을 STRING으로 반환한다. |
